# GitHub Notes

## Repository
A repository stores code, documentation, issues, pull requests, and project history.

## Commit
A commit is a saved snapshot of changes.

## Branch
A branch lets me work on changes without affecting the main version.

## Pull Request
A pull request is a request to merge changes from one branch into another.

## GitHub Actions
GitHub Actions can automatically run tests, checks, or deployment steps.
